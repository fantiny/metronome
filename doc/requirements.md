# 项目需求

## 目标
- 实现一个 Flutter 节奏器，帮助用户在演奏乐器时稳定打拍子。
- 界面采用现代简约扁平化风格。

## 核心功能（首版）
- 开始/停止节拍
- BPM 调节（范围 40–208，默认 120）
- 4/4 固定拍号
- 音色切换
- 音量滑块 + 静音
- 节拍视觉反馈（中心圆脉冲）
- 设置页：默认 BPM/音色/音量/静音/阻止休眠开关（默认开启）
- 设置持久化
- 防止设备休眠：仅在播放时生效

## 约束与偏好
- 架构：Clean Architecture（模块化、接口化、易扩展）
- 节拍稳定：优先音频排程
- 平台：iOS + Android
